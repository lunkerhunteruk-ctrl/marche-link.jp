# オンボーディングの表記・UI変更

## 1. サブドメイン欄の表記変更

### 変更内容

「サブドメイン（URL用）」を、どのショップでも分かりやすい表現に変更します。

### 1. ラベル（項目名）

- **変更前**: `サブドメイン（URL用） *`
- **変更後**: `お店のページアドレス（○○.marche-link.jp の ○○ の部分） *`

### 2. 入力欄のプレースホルダー

- **変更前**: `例: tanaka → tanaka.marche-link.jp`
- **変更後**: `例: tanaka と入れると tanaka.marche-link.jp になります`

### 3. 説明文（ラベル下の小さな文字）

- **変更前**: `英小文字・数字・ハイフンのみ。予約語（admin, www 等）は不可。`
- **変更後**: `半角英小文字・数字・ハイフンのみ。admin, www は使えません。`

---

## 反映方法

### A. コンポーネントを使う場合

`components/onboarding/onboarding-step-basic.tsx` に上記の表記でステップ1用コンポーネントがあります。  
`app/admin/onboarding/page.tsx` でステップ1の部分をこのコンポーネントの利用に差し替えてください。

### B. 既存ページで文言だけ変える場合

`app/admin/onboarding/page.tsx` 内で、ステップ1のフォームがインラインで書かれている場合は、次のように置き換えてください。

1. `サブドメイン（URL用）` → `お店のページアドレス（○○.marche-link.jp の ○○ の部分）`
2. `例: tanaka → tanaka.marche-link.jp` → `例: tanaka と入れると tanaka.marche-link.jp になります`
3. `英小文字・数字・ハイフンのみ。予約語（admin, www 等）は不可。` → `半角英小文字・数字・ハイフンのみ。admin, www は使えません。`

## 2. 任意項目の記入を促すUI

「（任意）」だけだとスキップされがちなため、**「できるだけ記入するとイメージ通りのデザインに」** と促すUIに変更しています。

### 追加したもの

- **共通メッセージ**（各ステップの任意項目の直前に表示）  
  「できるだけ記入すると、イメージに近いショップデザインに仕上がりやすくなります。」  
  （青系の背景の注意書きボックス）

- **ラベル表記の変更**  
  - `（任意）` → **（任意・記入するとデザインの精度が上がります）** または **（任意・記入推奨）**

### コンポーネント

- `components/onboarding/optional-fields-note.tsx` … 共通メッセージ
- `components/onboarding/onboarding-step-basic.tsx` … ステップ1（上記メッセージ＋任意ラベルを反映済み）
- `components/onboarding/onboarding-step-concept.tsx` … ステップ2（コンセプト・ターゲット）
- `components/onboarding/onboarding-step-design.tsx` … ステップ3（デザイン希望）

### 既存ページで文言だけ変える場合

`app/admin/onboarding/page.tsx` でステップ2・3がインラインの場合は、次を適用してください。

1. **各ステップの任意項目ブロックの直前に**、次の1行を表示するボックスを追加する：
   - 「できるだけ記入すると、イメージに近いショップデザインに仕上がりやすくなります。」

2. **ラベル**  
   - `（任意）` を **（任意・記入するとデザインの精度が上がります）** または **（任意・記入推奨）** に置き換える。
